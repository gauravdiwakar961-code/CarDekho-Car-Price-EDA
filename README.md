# CarDekho-Car-Price-EDA
# 🚗 Car Price Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a used car dataset to identify the key factors that affect car selling prices. The analysis includes data cleaning, visualization, correlation analysis, and business insights using Python.

---

## 🎯 Objective

- Understand the dataset
- Clean the data
- Analyze numerical and categorical features
- Find relationships between variables
- Generate business insights through visualizations

---

## 📂 Dataset

- Dataset Name: CarDekho Car Price Dataset
- Records: 301
- Features: 9

---

## 🔍 Key Insights

- Petrol cars are the most common in the dataset.
- Selling Price is strongly related to Present Price.
- Older cars generally have lower selling prices.
- Cars with fewer kilometers driven tend to have better resale value.
- Manual transmission cars are more common than automatic cars.

---

## 💼 Business Recommendations

- Focus on newer cars for higher resale value.
- Maintain lower vehicle usage to improve resale price.
- Consider Present Price as a major factor while estimating resale value.
- Fuel type and transmission significantly influence selling price.
